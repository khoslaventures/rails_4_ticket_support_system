Rails 4 Ticket Support System

http://stefanosioannou.com

Instructions

* git clone https://github.com/stefioan/rails_4_ticket_support_system.git

* bundle install

* rake db:migrate

* Ready to use, launch the rails server locally to test.


Feel free to improve and report issues/bugs!


Notice: This Rails 4 Ticket Support System is intended for guidance and educational purposes. If you wish to use it in one of your Rails apps keep in mind that It needs heavy refactoring and integration.
